JavaScript

JavaScript

We follow the community-driven Airbnb Style Guide, with the following exceptions:

Rule Setting
no-console Not enforced
arrow-body-style Not enforced
quote-props "consistent-as-needed"
comma-dangle "always-multiline"
max-len Not enforced

General Best Practices at Dynamit

  • Vanilla JavaScript
  • use axios for ajax
  • For carousels/sliders, use Flickity
  • For browser cookie getting/setting, use js-cookie
  • write in modules
  • prefer libraries to full-blown frameworks
  • write modern javascript when you can
  • keep bundles small (< 50k), consider lazy-loading pages / components (or chunking with webpack)
  • Thoroughly document code use JSDoc conventions
  • Full JavaScript Guidelines
